# 작업 수칙 (R&D 시험 결과 보고서)

- 측정값은 입력 자료에 있는 값만 사용하고, 없는 값은 "확인 필요"로 표시한다.
- 자료에 없는 파생 계산값(초과율 % 등)을 원본 수치처럼 적지 않는다. 적을 경우 산출 근거를 함께 명기한다.
- 판정: 기준 초과 = "불합격", 기준 이내 = "합격", 기준이 자료에 없으면 "확인 필요".
- 보고서 문체는 "~함" 체, 단위는 SI 우선.
- 전문 약어는 처음 나올 때 풀어 쓴다(예: FFT = 고속 푸리에 변환).
- 원본 측정 데이터는 수정하지 않는다.

## 용어집
- 피로수명: 반복 하중에서 파손까지의 수명 (fatigue life)
- 불균형량: 회전체 질량 불균형 정도, 단위 g·mm
- NVH: 소음(Noise)·진동(Vibration)·불쾌감(Harshness)
